# QR Code Page

This is a simple web page that displays a QR code and some descriptive text. 
Users can scan the QR code to quickly access a linked resource.

## Features
- QR code display
- Clean layout
- Responsive design

## Technologies Used
- HTML
- CSS

## How to Use
Open the page in a browser and scan the QR code with your phone.

- index.html
